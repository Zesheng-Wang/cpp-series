# 能力目标
## 1. 变量和常量
- 创建变量的时候应该进行初始化
- 练习 : 交换连个变量的值
- const关键字
## 2. 变量命名规则
- Snake Case(小写字母来命名，下划线连接)
- Pascal Case(大写字母开头，每个单词首字母大写)
- Camel Case(小写字母开头，其后每个单词首字母大写)
- Hungarian Notation(匈牙利写法，以变量类型明开头, 其后使用Pascal命名法则，例
如 **int iFileSize;**)
## 3. 数学表达式
- +
- -
- *
- /(整数除法结果返回商)
- % 返回余数
- +=
- -=
- *=
- /=
- ++ : 自增
- \-- : 自减
## 4. 输入和输出
- cin
- cout
## 5. 使用标准库

## 6. 注释
